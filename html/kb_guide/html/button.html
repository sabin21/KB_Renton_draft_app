버튼
박스와 텍스트 타입으로 구성되어 있다.
박스타입
사용자의 행동을 결정하는 주요 요소로 수행하고자 하는 내용을 미리 인지시키고 수행할 수 있도록 한다.
버튼은 수행하게 될 작동 내용에 대하여 안내하는 레이블(Label)과 버튼의 작동 여부를 나타내는 백그라운드(Background)로 구성한다.
클래스 설명
btn-ty1 긍정의 표현, 사용자 행동 유도, 우선순위 높은 내용, 좌우 여백없이 와이드하게 사용
border : none / background-color : #fc0
btn-ty2 border : #bbb / background-color : #fff
btn-ty3 border : none / background-color : #ddd
btn-ty4 border: none / background-color : #eff1f4

텍스트타입
하이퍼텍스트 버튼은 텍스트와 함께 사용되는 형태에 따라 텍스트 적용형, 텍스트+밑줄 적용형, 아이콘 적용형, 혼합형으로 구분한다.
