레이아웃 순서나 태그 구조는 원하는대로 작성하되, data-toggle-btn 과 data-toggle-contents 의 값은 동일해야 합니다.
<button type="button" data-toggle-btn="toggle1" aria-selected="true" title="선택됨">개인사업자</button> <button type="button" data-toggle-btn="toggle2">법인사업자</button> <div data-toggle-contents="toggle1"> 개인사업자 내용 </div> <div data-toggle-contents="toggle2" style="display:none"> 법인사업자 내용 </div>
