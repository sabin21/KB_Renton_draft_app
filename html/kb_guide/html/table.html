테이블
데이터 테이블은 기초 데이터를 체계적으로 정리하여 전달하는 역할을 한다.
기본형
thead + tbody > th + td
클래스 설명
tbl-wrap div로 테이블을 감싸준다.
tbl-common tbl-wrap 내의 table 태그에 입력
<div class="tbl-wrap"> <table class="tbl-common"> <!-- 접근성 : caption에 th값 출력 --> <caption>제목(th,th,th,...)</caption> <colgroup> <col><col style="width:40%"><col style="width:40%"> </colgroup> <thead> <tr> <th scope="col">열 1</th> <th scope="col">숫자</th> <th scope="col">문자</th> </tr> </thead> <tbody> <tr> <th scope="row">문자</th> <td class="tar">우측정렬</td> <td>중앙정렬</td> </tr> <tr> <th scope="row">문자</th> <td class="tar">3,500원</td> <td>문자</td> </tr> </tbody> </table> </div>
가로형
tbody > th + td
클래스 설명
tbl-wrap div로 테이블을 감싸준다.
tbl-common tbody tbl-wrap 내의 table 태그에 입력 / 가로형 테이블에 tbody 클래스 추가
<div class="tbl-wrap"> <table class="tbl-common tbody"> <!-- 접근성 : caption에 th값 출력 --> <caption>제목(th,th,th,...)</caption> <colgroup> <col style="width:80px"><col style="width:"> </colgroup> <tbody> <tr> <th scope="row">문자</th> <td>내용이 2줄이상 길어질 경우 내용이 2줄이상 길어질 경우 내용이 2줄이상 길어질 경우 내용이 2줄이상 길어질 경우 </td> </tr> <tr> <th scope="row">숫자</th> <td class="tar">100,000,000,000,000원</td> </tr> </tbody> </table> </div>
세로형
thead + tbody > td
클래스 설명
tbl-wrap div로 테이블을 감싸준다.
tbl-common tbl-wrap 내의 table 태그에 입력
<div class="tbl-wrap"> <table class="tbl-common"> <!-- 접근성 : caption에 th값 출력 --> <caption>제목(th,th,th,...)</caption> <colgroup> <col style="width:33.3%"><col style="width:33.3%"><col style="width:"> </colgroup> <thead> <tr> <th scope="col">열 1</th> <th scope="col">숫자</th> <th scope="col">문자</th> </tr> </thead> <tbody> <tr> <td>데이터</td> <td class="tar">우측정렬</td> <td>중앙정렬</td> </tr> <tr> <td>데이터</td> <td class="tar">3,500원</td> <td>문자</td> </tr> <tr> <td>데이터 2줄 이상 출력될 경우 데이터 2줄 이상 출력될 경우</td> <td class="tar">3,500원</td> <td>문자</td> </tr> </tbody> </table> </div>
